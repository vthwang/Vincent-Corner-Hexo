HTML5+CSS3 快速學習自我挑戰 Day3

2017-11-30
CSS, HTML
Symbols count in article: 2.7k Reading time: 16 mins.

殺手級的網站

製作 Features Section

  1. 下載 icons，然後將 css 複製到 vendors/css、fonts 複製到 vendors/fonts
  2. 在 index.html 引入 ionicons
    <link rel="stylesheet" type="text/css" href="vendors/css/ionicons.min.css">
  3. 在 header 下方加入 Feature Section
  1. 在 index.html 的 col 部份新增 box 和 icon 部分新增 icon-big 的 class
  1. 在 style.css 的 REUSABLE COMPONENT 新增 section 和 box
  1. 在 style.css 的 HEADINGS 將共同的 css 取出來，並新增 h1 h2 h3
  1. 在 style.css 的 PARAGRAPHS 將共同的 css 取出來
  1. 在 style.css 的 ICONS 新增 icon-big

建立 Favorite Meals Section

  1. 在 index.html 新增一個 Favorite Meals Section
  1. 在 style.css 的 MEALS 新增樣式
  1. 在 style.css 的 MEALS 讓中間文字說明和下方段落有間距

建立 How-it-works Section

  1. 在 index.html 新增 How-it-works Section
  1. 在 style.css 的 HOW IT WORKS 新增樣式
  1. 在 style.css 的 BASIC SETUP 新增 clearfix
  1. 在 index.html 的 meals-showcase 後面加上 clearfix meals-showcase clearfix

建立 Cities Section

  1. 在 index.html 新增 Cities Section
  1. 在 style.css 的 CITIES 新增樣式
  1. 在 style.css 的 ICONS 新增樣式
  1. 在 style.css 的 LINKS 新增樣式
  1. 在 style.css 的 HOW IT WORKS 把 LINKS 底線移除

建立 Customer Testimonials Section

  1. 把 resources/img/back-customers.jpg 放到 resources/css/img/back-customers.jpg
  2. 在 index.html 新增 Customer Testimonials Section
  1. 在 style.css 的 TESTIMONIALS 新增樣式
  1. 在 style.css 的 header 新增 background-attachment: fixed;，讓圖片在背景部分靜止不動，做成視差捲軸
