MongoDB，Mongoose 和 REST APIs (Todo API)

安裝 MongoDB 和 Mongoose

  1. 下載 MongoDB
  2. 將檔案解壓縮後，修改檔名為 mongo，然後移動到家目錄底下
  3. 在家目錄新增 mongo-data 的資料夾
  4. 啟動 server 並指定路徑./mongod --dbpath ~/mongo-data
  5. 新開分頁，執行 ./mongo
  6. 寫入資料庫 db.Todos.insert({text: 'File new node course'})
  7. db.Todos.find() ，搜尋資料庫會發現剛剛輸入的資料
  8. Robomongo

建立 NoSQL Vocabulary

  1. SQL 有 table 結構，NoSQL 則是使用物件的形式 (Collection)
  2. SQL 每一筆資料稱為 Row/Record，NoSQL 則稱為 Document
  3. SQL 每一個欄位稱為 Column，NoSQL 則稱為 Field

連接 MongoDB 並寫入資料

  1. MongoDB native
  2. mkdir node-todo-api
  3. npm init
  4. npm install mongodb@2.2.5 --save
  5. 新增 playground/mongodb-connect.js
  1. node playground/mongodb-connect.js
  2. 在 playground/mongodb-connect.js 插入一行
  1. node playground/mongodb-connect.js
  2. 在 playground/mongodb-connect.js 新增 Users 資料表，且新增 fields (name, age, location)
  1. node playground/mongodb-connect.js

ObjectId

  1. 將 const MongoClient = require('mongodb').MongoClient; 取代為 const { MongoClient, ObjectID } = require('mongodb');

取得資料

  1. 直接在 Robo 裡面的 Todos 新增 document
  1. 複製 mongodb-connect.js 並新增為 mongodb-find.js
  1. node playground/mongodb-find.js
  2. db.collection('Todos').find({completed: false}).toArray().then((docs) => {
  3. 這樣就只會出現 false 的選項了 node playground/mongodb-find.js
  4. db.collection('Todos').find({_id: new ObjectID('59bb791adf952c24d5ae60b7')})
  5. 也可以用 id 的方式搜尋 node playground/mongodb-find.js
  6. 用計數的方式算出東西

Deleting Document

  1. 一次刪除很多 deleteMany
  1. 一次刪除一個 deleteOne
  1. 選擇一個然後刪除，如果有多個一樣的，會自動選擇第一筆找到的 findOneAndDelete

更新資料

  1. 更新 Todos，set 改變值，returnOriginal 設為 false 就是回傳值的時候，傳回更新後的值，預設為 true
  1. 更新 Users，inc 設為 1 就是把值增加 1

設定 Mongoose

  1. Mongoose 官方網站
  2. npm i mongoose@4.5.9 --save
  3. 使用 Mongoose 寫入資料到 MongoDB
  1. 再寫入別的資料

Validators，Types 和 Defaults
