Nodejs 快速學習自我挑戰 Day8

2017-09-19
Nodejs
Symbols count in article: 1.8k Reading time: 10 mins.

MongoDB，Mongoose 和 REST APIs (Todo API)

部署 API 到 Heroku

  1. 安裝 Heroku CLI
    brew install heroku/brew/heroku
  2. 在 server/server.js 新增 port 的設定
    const port = process.env.PORT || 3000;
  3. 修改 app.listen
  1. 在 package.json 新增 start 指令，並新增 engine
  1. heroku create
  2. heroku addons:create mongolab:sandbox
  3. 可以取得 MongoDB URI heroku config
  4. 修改 server/db/mongoose.js
    mongoose.connect(process.env.MONGODB_URI || 'mongodb://localhost:27017/TodoApp');
  5. git push heroku master
  6. 如果應用發生問題，可以用以下指令確認 heroku logs

Postman 環境

  1. 進去 Postman，右上角有 Management Environment，點進去之後，新增 Todo App Local 和 Todo App Heroku，分別設定 url 為 localhost:3000 和 Heroku 的網址
  2. 然後將網址都改為 的變數代替

刪除 Resource - DELETE /todos/:id

  1. 新增 playground/mongoose-remove.js
  1. 在 server/server.js 新增 delete route
  1. 使用 Postman 測試，並儲存

測試 DELETE /todos/:id

  1. 新增第一個測試，最後的部分是驗證資料是否被刪除
  1. 基本上做法跟 GET 差不多，複製 code，然後把 .get 改成 .delete 即可

更新 Resource - PATCH /todos/:id

  1. npm i --save lodash@4.15.0
  2. 修改 server/server.js，引入 lodash，將其它引入的資料用 const。
  1. 新增 patch，如果 completed = true，寫入日期，如果沒有，把 completed 設為 false 且 completedAt 設為 null
  1. 用 Postman 測試，傳入 JSON，本地端測試完，再用 Heroku

測試 PATCH /todos/:id

  1. PATCH test 的設計邏輯
  1. 測試修改 completed 為 true
  1. 測試修改 completed 為 false

創建測試 Database

  1. 修改 package.json
  1. 修改 server/db/mongoose.js
    mongoose.connect(process.env.MONGODB_URI);
  2. 新增 server/config/config.js
  1. 在 server/server.js 引入 config
    require('./config/config');
  2. node server/server.js，會進入 development 環境
  3. npm test，會進入 test 環境，會再新建一個資料庫

安全和認證

建立 User Model

  1. npm i validator@5.6.0 --save
  2. 修改 server/models/user.js
  1. 修改 server/server.js

JWTs 和 Hashing

  1. npm install crypto-js@3.1.6 --save
  2. 新增 playground/hashing.js
  1. 驗證 data 如果不同會跳出資料改變的訊息
  1. npm i jsonwebtoken@7.1.9 --save
  2. JWT 官方網站
  3. 使用 jwt 來進行驗證

產生 Auth Tokens 和 Setting Headers

  1. 修改 server/models/user.js
  1. 修改 server/server.js
