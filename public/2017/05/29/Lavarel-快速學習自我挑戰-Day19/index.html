Application 留言功能 Part II

顯示留言

  1. 在 admin/posts/index view 新增文章連結
    <td><a href=" {{route('home.post', $post->id)}}">view post</a></td>
  2. 在 admin/posts/index view 新增評論連結
    <td><a href=" {{route('admin.comments.show', $post->id)}}">View Comments</a></td>
  3. 新增 admin/comments/show view
  4. 在 PostCommentsController 新增 show function
  1. 修改 Post model

在文章中顯示留言

  1. 新增 Auth 到留言區塊，讓登入的使用者才可以留言
  1. 修改 AdminPostsController 的 post function
  1. 新增 posted comments 到 post view

回覆留言功能

  1. 新增 routes
  1. 在 CommentRepliesController 新增 CreateReply function
  1. 在 CommentReply model 新增 'photo' 讓欄位可寫入
  2. 新增 reply comment 到 post view
  1. 在 layouts.blog-post view 生成 script 區塊
    @yield('scripts')
  2. 新增 div
  1. 在 resources/assets/scss/app.css 修改樣式，新增 .comment-reply {display: none;}，並用gulp編譯。
  2. 新增 scripts 區塊

在主管理介面顯示回覆

  1. 複製 comments/show view 到 comments/replies/show view，將 comment 改為 reply。
  2. 在 admin/comments/index view 新增
    <td><a href=" {{route('admin.comments.replies.show', $comment->id)}}">View Replies</a></td>
  3. 在 CommentRepliesController 新增 show function
  1. 在 CommentRepliesController 新增 update function
  1. 修改 post view
  1. 在 CommentRepliesController 新增 destroy function
