Pretty URL

  2. require 套件 composer require cviebrock/eloquent-sluggable 4.1
  3. 在 config/app.php 新增 provider
    Cviebrock\EloquentSluggable\ServiceProvider::class,
  4. 在 Post model 新增
  1. 新增 migration php artisan make:migration add_slug_to_posts_table --create=posts
  2. 修改 add_slug_to_posts_table
  1. 將資料匯入資料庫 php artisan migrate
  2. 修改 admin/posts/index view 的檢視貼文欄位
    <td><a href="{{" {{route('home.post', $post->slug)" }}}}">View post</a></td>
  3. 修改 AdminPostsController 的 post function

升級

  5. 把 app/Providers/EventServiceProvider 的 boot function 的變數拿掉
