使用 Socket.io 的即時 Web Apps

Mustache.js

  1. 下載 Mustache 最新版，儲存到 public/js/libs/mustache.js
  2. 在 public/index.html 引入 <script src="js/libs/mustache.js"></script>
  3. 在 public/index.html 加入 template
  1. 在 public/js/index.js 使用 Mustache 來顯示 template
  1. git push heroku master

Autoscrolling

  1. 在 public/js/index.js 新增 scrollToBottom function
  1. 在 public/js/index.js 的 newMessage 和 newLocationMessage 新增 scrollToBottom
    scrollToBottom();

新增一個 join page

  1. 將本來的 index.html 和 index.js 改為 chat.html 和 chat.js
  2. 新增 public/index.html

傳送 Room 資訊

  1. deparam function
  2. 把檔案下載到 public/js/libs/deparam.js
  3. 在 public/chat.html 引入 <script src="js/libs/deparam.js"></script>
  4. 在 public/js/chat.js 新增 emit
  1. 新增 server/utils/validation.js
  1. 在 server/server.js 引入 const {isRealString} = require('./utils/validation');
  2. 在 server/server.js 新增 socket.on 取得資訊，並做驗證
  1. 新增 server/utils/validation.test.js
    • 引入 isRealString
    • 測試一：拒絕不是 string 的值
    • 測試二：拒絕只有空白的 string
    • 測試三：接受沒有空格的 string
  2. 新增 server/utils/validation.test.js

Socket.io Rooms

  1. 在 server/server.js 新增 socket.join，然後把上面的 newMessage 放到下面，動態產生使用者加入的訊息

使用 ES6 classes 來儲存用戶

  1. 新增 server/utils/users.js
  1. node server/utils/users.js 就會出現 Andrew 25
  2. 在 class 新增 getUserDescription
  1. 新增變數，然後讀出變數
  1. console.log(description); 就會顯示出 Andrew is 25 year(s) old.
  2. 在 server/utils/users.js 新增 Users class
  1. 新增 server/utils/users.test.js
  1. npm test
  2. 在 server/utils/users.test.js 丟 data 進去
  1. npm run test-watch
  2. 在 server/utils/user.js 新增 getUserList
  1. 在 server/utils/user.test.js 新增 getUserList 測試
  1. 在 server/utils/user.js 新增 getUser
  1. 在 server/utils/user.test.js 新增 getUser 測試
  1. 在 server/utils/user.js 新增 removeUser
  1. 在 server/utils/user.test.js 新增 removeUser 測試

Wiring up User List

  1. 在 public/js/chat.js 新增 updateUserList
  1. 在 server/server.js 引入 const {Users} = require('./utils/users');
  2. 在 server/server.js 新增 users var users = new Users();
  3. 在 server/server.js 的 connect 部分，當連線時，先 removeUser，然後再重新 addUser，最後將 userList 傳送到前端頁面
  1. 在 server/server.js 的 disconnect 部分，踢除 user，如果 user 存在，就更新 UserList，然後在前端頁面顯示 user 離開的訊息
  1. 修改 public/js/chat.js 的 updateUserList，把 user 顯示在前端
